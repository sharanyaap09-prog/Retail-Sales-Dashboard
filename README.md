# 📊 Retail Sales Dashboard

## Project Overview

This project is an interactive Retail Sales Dashboard built using Streamlit, Pandas, and Plotly. It helps analyze retail sales data through visualizations, KPIs, and filters.

The dashboard allows users to monitor sales performance, profit, customer behavior, and regional growth.

---

## Features

- 📈 Monthly Sales Trend Analysis
- 💰 Revenue and Profit KPIs
- 🛒 Average Order Value (AOV)
- 📦 Total Orders Tracking
- 🏷️ Sales by Category
- 👥 Top 10 Customers Analysis
- 🌍 Region-wise Sales Comparison
- 🔍 Interactive Filters
  - Date Range
  - Region
  - Category
  - Segment
- 📋 Filtered Data Table

---

## Installation

### Clone Repository

```bash
git clone https://github.com/your-username/Retail-Sales-Dashboard.git
```

### Navigate to Project Folder

```bash
cd Retail-Sales-Dashboard
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run Application

```bash
streamlit run app.py
```

The dashboard will open in your browser at:

```
http://localhost:8501
```
